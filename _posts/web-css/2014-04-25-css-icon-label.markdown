## 字体图标

字体图标在浏览器解读时，会被解读成文字，所以能使用文字的所有样式对其进行设定

```
<i class="fonticon icon-..."></i>
```

## label标签

当你点击label标签时，浏览器就会自动将焦点转到和标签相关的表单控件上 

```
<label for="uname">姓名：</label>
<input type="text" id="uname"/>
```
